## 오늘한 일

- window 관리자 연결

### **오늘의 정리**

- 서울로 가게 되면서 원래 사용하던 데탑을 부모님이 사용하실 수 있도록 비밀번호를 삭제하려다가... admin 계정 없애도 되는 줄 알고 일반 유저로 변경했다가 대참사가 발생했다
- 관리자 권한으로 실행 자체가 불가능해지는 문제가 생겼고, 다시 설정을 하려고 했지만 관리자 계정(?)자체가 없어짐으로써 수정자체를 할 수 없게 되었다.

##### 해결 방법

- 일단 마이크로소프트에 올라온 질문들에 해당하는 내용들을 다 적용해보았으나, 모두 관리자 계정이 있다는 전제하에 설정이 가능했고, 안전모드에서 실행하는 방법의 경우 내 컴퓨터는 관리자 계정이 없어진 상태라, 올바른 비밀번호를 입력해도 틀렸다고 뜨는 상황에 이르렀고, <br/> 안전모드에 접근하지 못할 경우 윈도우를 새로 까는 방법 밖에 없다는 내용들 뿐이었다.

- 그러다가 발견한 새로운 방법인 [microsoft](https://answers.microsoft.com/ko-kr/windows/forum/all/윈도우-10에서/8ee4e45e-20bb-4fea-b309-f858a88261f4) 여기서 말하는 shift를 누른 채로 다시 시작을 누르는게 정답이었다.
- 자연스럽게 안전모드로 진입할 수 있었고, 여기서 administrator 계정을 만들어주고 원 방법대로 진행을 하면 관리자 계정이 생겨나게 세팅할 수 있게 되었다.
